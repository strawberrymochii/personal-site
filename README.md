Personal Site

This is a personal site that I made for my best food pics and my cutest cat pics! This was the first project I made using Astro, which made for quite a learning curve. Through working on this website, I got to learn the features of Astro, including its components and layouts. Figuring out the correct syntax for the image paths was pretty challenging, since they have to be very exact, and I couldn't just copy the path from my folder. Overall, I really enjoyed working on this project, and I really like how easy it was to start Astro. I'm really looking forward to adding more to my website!
